# `heapq` - For Min-Heaps & Priority Queues
| Function        | Purpose                |
| --------------- | ---------------------- |
| `heappush()`    | Insert into heap       |
| `heappop()`     | Remove smallest        |
| `heapify()`     | Convert list to heap   |
| `heappushpop()` | Push and pop in one go |
| `nlargest()`    | Top `n` biggest        |
| `nsmallest()`   | Top `n` smallest       |
